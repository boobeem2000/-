# Recurrent Biz Colleage

ビジネスコーチングとAI開発のプロジェクト

## プロジェクト概要

このリポジトリには、リカレントビジネス・カレッジの実践講座に関する資料やAI開発プロジェクトが含まれています。

## 構成

- `coaching_ai.py` - コーチングAIアプリケーション
- `create_excel.py` - Excelファイル生成スクリプト
- `docs/` - ドキュメント
- `FS/` - フロントセールス関連資料
- `人生の輪/` - 人生の輪ワークシート
- `価値観を明確化するワーク/` - 価値観明確化ワークシート
