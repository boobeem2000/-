# Recurrent Biz Colleage

ビジネスコーチングとAI開発のプロジェクト

## プロジェクト概要

このリポジトリには、リカレントビジネス・カレッジの実践講座に関する資料やAI開発プロジェクトが含まれています。

## 📁 フォルダ構造と使い方

### 🎯 ルートファイル

#### Pythonスクリプト
- **`coaching_ai.py`** - 事実と解釈を分ける力を育てるコーチングAI
  - 使い方: `python coaching_ai.py` で実行
  - 対話形式で気づきをサポートするAIアプリケーション

- **`create_excel.py`** - Atsuko式スピーキング再起動ゲーム用Excelファイル生成
  - 使い方: `python create_excel.py` で実行
  - Excelワークブック（Dashboard, Mission Log, Instructions）を自動生成

#### ゲーム関連ファイル
- **`Atsuko_Speaking_Restart_Game.html`** - スピーキング再起動ゲーム（HTML版）
- **`Atsuko_Speaking_Restart_Game_*.csv`** - ゲームのデータファイル（Dashboard, Instructions, Mission Log）

#### PDF資料
- **`CheckSheet(arikata).pdf`** - チェックシート
- **`customers_voice.pdf`** - 顧客の声
- **`recurrent_recipe.pdf`** - リカレントレシピ
- **`振り返りシート.pdf`** - 振り返りシート

## 🚀 使い方

### Pythonスクリプトの実行

```bash
# コーチングAIを実行
python coaching_ai.py

# Excelファイルを生成
python create_excel.py
```

### ファイルの管理

- 新しいクライアントの資料は、適切なフォルダに保存
- ファイル名には日付やクライアント名を含めることを推奨
- 個人情報が含まれるファイルは適切に管理

---

## 📝 注意事項

- 個人情報が含まれるファイルは、適切に管理してください
- `.gitignore`で機密情報を除外しています
- 大容量ファイル（動画など）はGitHubにアップロードしないでください
